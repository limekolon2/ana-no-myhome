# 穴の中のマイホーム

地面を掘って、地下に自分だけの秘密基地を作るゲーム。

## コンセプト

プレイヤーは地上からスタートし、足元の土を掘り進めて地下空間を作っていく。掘った空洞に家具を設置して、自分だけの地下住居を充実させていく。家具を3つ以上置くと、マップ最深部の岩盤を掘り抜けるようになる。岩盤を砕くと、下から光が差し込んでくる。その光の穴に飛び込むと——世界がひっくり返り、地上へ出るエンディングを迎える。

## 操作

- 矢印キー / WASD：移動・掘削（移動先が土や石なら自動で掘る）
- スペース：家具の設置 / 撤去
- F：家具の切り替え
- R：リセット
- タッチ操作にも対応（十字パッド＋ボタン）

## ゲームの流れ

1. 地上の草原からスタート
2. 下方向に掘って地下空間を広げる
3. 家具を選んで設置し、部屋を作り込む
4. 家具を3つ以上設置すると、最深部の岩盤がうっすら光り始める
5. 岩盤を下に掘ると光の穴が開く
6. 光の穴に入ると画面が上下反転し、エンディングへ

## 家具一覧（12種）

### 1マス家具
- はしご：上下移動用。はしごがないと落下する
- ランプ：周囲3マスを暖色で照らす
- 暖炉：周囲4マスを照らす。炎がゆらめくアニメーション付き
- 大きな植物：鉢植えの観葉植物
- 宝箱：装飾用の宝箱
- 壁の地図：壁に貼る冒険の地図

### 2マス家具
- 本棚：色とりどりの本が並ぶ
- ベッド：枕と青い毛布つき
- 作業台：ハンマーや万力が乗った工作台
- 水槽：熱帯魚が泳ぐ。周囲2マスを青白く照らす
- ペルシャ絨毯：金の装飾パターン入り
- 食卓：パンとコーヒーとキャンドル付き

## 主な特徴

- **掘削と建築の一体感**：移動するだけで掘れる。掘った空間にそのまま家具を置ける
- **ライティングシステム**：ランプ・暖炉・水槽が周囲のマスに光を落とす。暖炉は暖色、水槽は寒色。家具が増えると空間の雰囲気が変わっていく
- **重力とはしご**：はしごがない場所では落下する。縦穴の移動にははしごが必要
- **反転エンディング**：地下を掘り進めた先で光を見つけ、世界がひっくり返って地上に出る演出。5行の詩と風のパーティクルで締めくくる

## 技術仕様

- React + Vite
- Canvas 2D描画（RENDER_SCALE 2倍）
- マップサイズ：60×56タイル（タイル20×20px、表示40×40px）
- 視野：18×11タイル
- 家具画像は `/public/furniture/` にPNGで配置。コード内のピクセルアートがフォールバック
- Web Audio APIによる効果音（掘削音、足音、設置音）＋環境音（地下ドローン、地上の風）
- iPad タッチ操作対応
